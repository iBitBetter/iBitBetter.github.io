果核看图 — 极速看图，不止于快

不知道有没有平时喜欢拿着单反、微单出去一顿狂捏的兄弟？如果有，那你一定懂那种绝望：每次高高兴兴扫街回家，把 SD 卡插进电脑，面对里面两三千张、每张动辄几十上百兆的 RAW 格式原图，脑瓜子都是嗡嗡的。

选图，绝对是现代摄影最反人类的环节，没有之一。如果你头铁，非要用 Windows 自带的那个“照片”应用来看，那我敬你是个狼人。双击一张图，屏幕中央就开始转那个万恶的小圆圈，等个两三秒图片才从马赛克慢慢变得清晰。两千多张图这么选下来，鼠标还没点烂，人已经先脑溢血了。

后来我也折腾过不少办法。用 Lightroom 吧？太重了。为了挑出几十张能用的废片，非得先花半小时把全部照片导进去，还得听着电脑风扇狂转、等它慢吞吞地建预览，简直是杀鸡用牛刀。找其他第三方看图软件吧，速度是快了点，但打开一看，颜色全偏了——原本在相机屏幕上看着油润的色彩，在软件里灰得像盖了一层霾。

直到前阵子，有个搞商业摄影的哥们实在看不下去我这龟速工作流，往我微信上扔了个叫 “果核看图”（原名 MagicView）的安装包。

他说这玩意能秒开。我当时心里想的是：你扯吧，上百兆的中画幅 RAW，就算是神仙来了也得加载一下吧。

结果安装完一试，我当场在电脑前爆了句粗口。这东西是真的邪门，几十上百兆的图，双击的瞬间，啪！直接就糊你脸上了。没有过渡动画，没有加载条，没有任何让你哪怕眨一下眼的缓冲时间。那种“你点多快，它就出多快”的跟手感，爽得让人头皮发麻。习惯了这个速度再去用别的软件，你会觉得它们全都在磨洋工。

后来去他们官网看了眼才明白，这作者是个狠人。现在市面上大多软件为了省事，都喜欢套个网页壳（Electron）来做，臃肿得要死。这软件底层纯是用 C++ 一行行死磕出来的，把 CPU 的多核性能榨干了，就为了这零点几秒的极限速度。

更让我服气的是它的色彩还原。索尼的 ARW、佳能的 CR3、尼康的 NEF，丢进去出来的颜色，基本跟在 Lightroom 里看到的标准色彩一模一样。对咱们这种经常要修图的人来说，不偏色、不断层，比什么花里胡哨的功能都强一百倍。

这软件还特别对我这种“软件洁癖患者”的胃口。界面极简，超窄边框，看图的时候连菜单栏都自动隐藏了，整个屏幕就只剩下你的照片。没有弹窗，没有广告，甚至官网还提供了一个绿色免安装版，解压就能用，绝对不往你电脑的注册表里塞垃圾。

当然了，天下没有完美的工具。这作者大概是个极致的技术偏执狂，为了保证这变态的加载速度，他直接放弃了老旧的电脑配置。目前这玩意儿只支持 Windows 10 和 11，而且要求你的 CPU 必须支持一些比较新的指令集，太老的古董机根本跑不起来。至于用 Mac 的兄弟，很遗憾，暂时没你们什么事。

但退一步想，这年头大家都想做大做全，什么功能都往里塞。能有个开发者愿意顶着挨骂的风险，放弃一部分市场，就把“在 Windows 上看图极快”这一件破事死磕到天花板，其实挺让人敬佩的。

这软件反正基础功能全免费。如果你平时也被成堆的 RAW 文件折磨，或者单纯就是受够了系统自带看图软件的墨迹，强烈建议去他们官网（pic.ghxi.com）搞个绿色版放电脑里。信我，只要体验过那种零延迟的爽感，你就再也回不去了。
