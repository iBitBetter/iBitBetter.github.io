### 如何实现多店铺共享面单取号

/logistics/newCreateOrder﻿ token字段传开通电子面单店铺的token，user_id字段传订单所在店铺的shopid或者传-1。（例：A店铺开通了电子面单，现在要用B店铺的订单获取单号，则token传A店铺的，user_id传B店铺的shopid或者-1）
