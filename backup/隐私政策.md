# 隐私政策

**生效日期**：2026年6月29日

## 我们收集的信息

iBitBetter（https://ibitbetter.github.io）是一个静态博客站点，托管于 GitHub Pages。

**我们不主动收集任何个人信息。** 本网站：

- 不使用 Cookie 追踪用户行为
- 不运行任何分析/统计脚本（如 Google Analytics）
- 不收集用户邮箱或注册信息
- 评论区由 utteranc.es 提供，评论数据存储在 GitHub Issues 中，受 GitHub 隐私政策约束

## 第三方服务

- **GitHub Pages**：托管平台，可能记录访问日志。详见 [GitHub 隐私声明](https://docs.github.com/cn/site-policy/privacy-policies/github-privacy-statement)
- **utteranc.es**：评论系统，基于 GitHub Issues，不独立存储数据
