前两天联调一个接口，对面甩过来一串挤成一坨的 JSON，睁大眼睛找半天上哪少了个括号。又赶上要解一段 Base64，手边偏偏没趁手的家伙。

这种时候人最容易犯懒——随手搜个在线格式化网站，把那串玩意儿往里一贴。贴完才后知后觉：这串里带着 token、带着用户字段，我刚把它明晃晃交给一个连名字都没听过的站了。

更别提上班处理 PDF。电脑上 Acrobat 压根没装，临时要合并两份合同、把那个 30 兆的扫描件压小点发出去，或者拆开一份锁了权限的文档。百度一搜"PDF 合并"，跳出来的全是花花绿绿的野站，点进去先弹三个广告，心里还得犯嘀咕：这文件里是客户资料，真要走它们的服务器绕一圈？

后来摸着个叫 [WorkTool](https://www.worktool.app/zh) 的站，好家伙，算是把这两桩心病一块儿治了。

它底下分了几摊：开发者工具、文档处理、图片处理、数据转换、职场效率、AI 助手，拢共九十好几个。我平时使的那几样，它全有。

先说写代码那头。JSON 格式化校验、Base64 编解码、URL 编解码，三个都在。关键是这类活儿天生就是浏览器里当场算的，基本就在你本地浏览器里算完，走不到别人服务器上——你贴一坨带 token 的 JSON 进去，它就在你本地给你排好版，不像那些"格式化完顺便存一份"的野鸡站，让人心里发毛。

再说 PDF。合并、拆分、压缩、解锁权限、加页码、旋转、裁剪，一应俱全。前阵子要把一份锁了打开密码的 PDF 卸掉限制，它直接给解了；30 兆的扫描件压完体积砍掉一大半，发邮件不卡了。重点是——不用装任何软件，网页点开就用，也不用注册。

它家图片压缩那条还明写着"无需上传服务器"。这年头肯把这句话拍在脸上的工具不多，起码说明人家在乎这个。

基础功能全免费，不注册也能使；要更高额度再升 Pro。普通人平时折腾那几下，免费档绰绰有余。

说真的，处理敏感文档、代码、图片，最怕的就是随手丢给来路不明的站。WorkTool 不一定样样都神，但至少你搜得到它、看得到它明说的隐私姿态，比随手点开一个广告弹屏的小站强太多。

下次再遇着要格式化 JSON、解 Base64、或者救个 PDF，别急着把文件交出去。去搜 WorkTool，网页开起来就完事。
