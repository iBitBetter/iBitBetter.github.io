果核看图（原 MagicView）是一款专注 Windows 平台的极速 RAW 图片查看器，首张加载 0.1 秒，支持 ARW、CR3、NEF、DNG、RAF、TIFF、HEIC、WebP 等 140+ 图片格式，纯净无广告，免费下载使用。

**快到什么程度？**

底层 C++，不是 Electron 套壳那种"还行吧"的快，是编译器层面就快的那种。自研解码引擎，多核 CPU 线程级优化，AVX-512 也支持。预热缓存一上，官方说 0.1 秒呈现。我试了，没吹牛。

100MB 的中画幅 RAW，点一下就出来了。没有进度条，没有转圈等待，就是"啪"地一下。用过这种速度再回去用别的看图软件，你会觉得它们都在磨洋工。

**40+ 格式，摄影师的万能钥匙**

支持 140 多种图片和 RAW 格式：索尼 ARW、佳能 CR3、尼康 NEF、DNG、HEIC、TIFF、WebP……能想到的基本都有。而且不只是"能打开"，主流相机厂商的私有格式做了原生级色彩还原，不是简单解个码糊弄你。

这点对摄影师太重要了。很多看图软件号称支持 RAW，打开一看色彩全偏，还不如直接看 JPEG 缩略图。果核看图在色彩还原上确实下了功夫。

**干净到不像 Windows 软件**

没广告，没捆绑，没弹窗，启动时只做一次轻量的版本检查。界面走沉浸路线，超窄边框，菜单智能隐藏，不看图的时候界面几乎不存在。这种克制，在 Windows 上太稀缺了。

还有免安装的绿色版——解压即用，不写注册表，塞U盘随身带。对有洁癖的用户来说，这比什么都重要。

**也有局限**

目前只支持 Windows 10/11，x64 架构且要 AVX2+。老电脑和 32 位系统跑不了，Mac 和 Linux 用户暂时没份。绿色版没有文件关联，得手动拖进去打开，也没自动更新。

但话说回来，这些限制反而说明开发者知道自己要做什么。与其做一个全平台通吃但处处妥协的产品，不如把一个平台做到极致。

**适合谁？**

摄影师，经常要快速浏览大量 RAW 文件；对 Windows 自带照片查看器忍了很久的人；受不了加载等待的急性子；喜欢绿色免安装、不碰注册表的软件洁癖患者。

果核看图不是功能最全的看图软件，但可能是 Windows 上最快的。它不替代 Lightroom 做后期，不替代 Bridge 做管理，就把"看图"一件事做到了当前平台的天花板。什么都想做大做全的时代，这种专注反而让人眼前一亮。
