比特币：一种点对点的电子现金系统

我们提出一种完全点对点的电子现金系统，允许在线支付直接由一方发起并支付给另一方，无需经过任何金融机构。通过工作证明链为交易记录打上时间戳，形成不可篡改的公开历史。只要诚实节点控制大多数 CPU 算力，攻击者在计算上就不可能篡改系统。

核心创新

  • 去中心化：无铸币厂、无银行、无中央权威
  • 工作证明：一个 CPU 一票，算力决定共识
  • 时间戳链：交易哈希成链，重做即不可能
  • 双重支付解决：公开广播所有交易，最长链即真相
  • 密码学安全：数字签名链确权，Merkle 树压缩存储

关键机制

  • 电子硬币定义为一条数字签名链
  • 每笔交易公开广播至所有节点
  • 节点将交易打包进区块，寻找工作证明
  • 找到后广播新区块，其他节点验证后开始下一轮
  • 难度动态调整以维持稳定出块速率
  • 区块奖励 + 交易费驱动经济激励

核心结论：诚实链与攻击链的竞争可建模为二项式随机漫步。当诚实节点掌握 >50% 算力时，攻击者追赶概率随确认区块数 指数级下降。6 个区块确认后，10% 算力攻击者的成功概率已降至 0.1% 以下 —— 这意味着比特币的安全性是计算保证的，而非制度保证的。
