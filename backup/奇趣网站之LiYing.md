## 项目背景与初衷

LiYing 是一个专为一般照相馆和自助打印店设计的自动化证件照处理程序，遵循 **AGPL-3.0** 开源协议，允许开发者与摄影工作者自由下载、二次开发或本地部署。项目名称“LiYing”源自开发者对其父母的感恩，旨在帮助他们更轻松地完成证件照后期处理工作。这种贴近实际需求的开发初衷，使得 LiYing 在功能设计上极具针对性和实用性。

LiYing 的核心目标是通过深度学习技术，解决传统证件照处理中的痛点，例如手动裁剪费时、背景更换复杂、排版不规范等。它不仅适合专业照相馆，也为个人用户提供了一个高效、便捷的解决方案。更重要的是，LiYing 支持完全离线运行，所有图像处理操作都在本地完成，避免了将敏感照片上传至云端的隐私风险。🔒

## 核心功能：智能化与高效并存

LiYing 的功能强大且全面，涵盖了证件照处理的全流程，以下是其核心亮点：

1. **人体与人脸自动识别**：LiYing 利用 **Yunnet** 进行人脸识别，结合 **yolov8n-pose** 模型实现人体检测，能够精准定位头肩区域，确保裁剪和排版的准确性。
2. **角度自动纠正**：通过智能算法，LiYing 可以自动调整照片角度，解决拍摄时因姿势倾斜导致的偏差，生成符合规范的证件照。
3. **背景自动更换**：支持 **RMBG-1.4/2.0** 模型，允许用户一键更换背景色（如白色、蓝色、红色等），无需手动抠图。
4. **灵活裁剪与排版**：LiYing 支持任意尺寸的证件照裁剪，并能根据用户需求自动排版，例如在一张 5 寸相纸上排版 3x3 的 1 寸照片，或 6 寸相纸上排版 4x2 的 1 寸照片。
5. **图像压缩与优化**：通过集成 **AGPicCompress** 项目，LiYing 支持 JPEG 和 PNG 格式的压缩，新增了 **size_range** 和 **target_size** 选项，确保输出照片在保持质量的同时满足特定文件大小需求。

这些功能使得 LiYing 在处理效率和输出质量上表现出色，尤其适合批量处理需求。无论是小型照相馆还是个人用户，都能轻松上手，快速生成合规的证件照。

## 安装与使用：简单易上手

LiYing 提供了多种运行方式，满足不同用户的需求。对于 Windows 用户（最低支持 Windows 7 SP1），可直接下载整合包，无需深入了解代码即可使用。整合包支持命令行（CLI）、批处理（BAT）和 WebUI 三种版本，其中 WebUI 版本通过浏览器访问（127.0.0.1:7860）提供图形化界面，操作更直观。

对于开发者或需要自定义功能的用户，可以通过以下步骤从源码构建：

1. 克隆项目：`git clone https://github.com/aoguai/LiYing`
2. 安装依赖：`pip install -r requirements.txt`
3. 下载模型（如 Yunnet、RMBG、yolov8n-pose）并放置在 `LiYing/src/model` 目录。
4. 运行命令行或 WebUI，例如：`python main.py --help` 查看参数，或 `run_webui.bat` 启动界面。

LiYing 还提供了详细的配置文件（如 `size_XX.csv` 和 `color_XX.csv`），用户可根据需求修改照片尺寸和背景色，灵活性极高。此外，项目支持 CI/CD 自动构建，确保版本更新及时，兼容性更强。
