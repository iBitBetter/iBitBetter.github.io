不知道你们平时怎么打 Emoji，我经常是在手机键盘上翻半天找不到想要的。在电脑上码字的时候更头疼，系统自带的表情面板有时候甚至呼不出来，好不容易搜到了，复制粘贴还经常出乱码。

昨天写东西的时候实在受不了，就在网上顺手找了个叫 SearchEmoji 的小网站（searchemoji.app），用着感觉意外地顺手。

<img src="https://ibitbetter.space/assets/images/SearchEmoji.webp" alt="SearchEmoji搜索引擎主页" width="600" height="300"/>

这东西做得很克制，网页里就是一个纯粹的搜索框。你想找什么表情，直接用中文搜就行。比如搜个“笑”，底下立马就给你列出苦笑、偷笑、捂脸笑这一大家子；搜个“猫”，各种猫相关的图标就全出来了。看中哪个，点一下就直接复制到了剪贴板，干脆利落。

虽然看起来就是个不起眼的小网页，但里面其实塞了快两千个表情，平时想随便逛逛也可以直接点分类标签翻。而且有很多做得挺细的地方，比如一键切换人物表情的肤色，日夜模式切换这些都有。

有个挺有意思的点是，它居然支持三十多种语言搜索，连印地语和孟加拉语都有，而且代码直接开源扔在 GitHub 上了。如果是做前端或者设计的，它里面那个“合格状态”筛选（看是不是符合规范的表情）估计会有点用。

整个网站干干净净，没广告也不让注册。反正我把它直接扔进浏览器收藏夹了，平时发推文、做图或者写东西需要配表情的时候，随手点开搜一下，比在键盘里划来划去省事多了。

对了，做这个网站的团队还顺手搞了个二十万量级的矢量图标库叫 Yesicon，有需要的设计师朋友也可以顺便去白嫖一下。
