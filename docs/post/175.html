菜谱里的"盐适量"逼疯了多少人？这神仙开源站把做饭写成了代码

前两天半夜饿得抓心挠肝。外卖软件滑到底，全是吃吐了的料理包。寻思着自己整点热乎的吧，一搜菜谱，脑血栓快气出来了："盐适量"、"油少许"、"大火翻炒至断生"。

神特么适量！少许到底是几克？断生到底是几分熟？

这帮写菜谱的，纯纯是给大厨看的，简直不给我们这种只会煮泡面的废柴留活路。

后来在网上一通乱挖，翻出个神仙破站：做饭指北。江湖人称《程序员做饭指南》。这玩意儿简直是炸厨房选手的亲爹。

根本不用纠结吃啥。脑子一抽想吃重口味，直奔"下饭菜"专区，宫保鸡丁、水煮肉片全给你码好。想嗦面？饺子面条烙饼全有，连和面加几毫升水、醒面精确到几分钟，标得明明白白。

最绝的是这帮理科生的硬核写法。

去他娘的"适量"这种玄学，人家全用克！几克盐、几毫升酱油、微波炉高火打多少秒，硬生生把做饭写成了跑代码。步骤全是一步步的条件判断加循环。你只要不是个瞎子，按顺序往下执行，照着抄绝对不翻车。我头一回炒了个番茄炒蛋，居然没糊，吃完甚至有点感动。

而且这破站开源全免费。四百多号懂代码的老哥在 GitHub 上疯狂给菜谱提 Bug、做更新。整个页面干净得连个开屏广告都没有，吊打外面那些满屏弹窗的流氓 APP。

下次半夜再饿得眼冒金星，别点那些恶心的外卖了。去搜 howtocook，挑个顺眼的。照着跑一遍代码，吃顿自己做的热乎饭，真特么舒坦。
